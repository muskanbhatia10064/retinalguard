# NeuroWatch — How to Run

## BACKEND (Terminal 1)
```
cd backend
pip install -r requirements.txt
python app.py
```
Backend runs at: http://localhost:5000

## FRONTEND (Terminal 2)
```
cd frontend
npm install
npm run dev
```
Website opens at: http://localhost:3000

## That's it! Open http://localhost:3000 in your browser.
- Click Normal / Kumbh Surge / Crush Scenario buttons
- Watch heatmap, risk gauge, venue map all update live
- Crush scenario → emergency voice announcement plays automatically
